Selectores CSS
-----------------------------------------------------------------------------------------------------------------------------
* Los selectores css son los unicos localizadores soportados por cypress
* Xpath no es soportado por cypress, antes se contaba con un plugin para poder utilizarlo pero quedÃ³ deprecado
-----------------------------------------------------------------------------------------------------------------------------
Sintaxis
-----------------------------------------------------------------------------------------------------------------------------
* cy.get(localizador)
-----------------------------------------------------------------------------------------------------------------------------
Tipos de selectores css
-----------------------------------------------------------------------------------------------------------------------------
* Etiqueta Id => #id
* Etiqueta Clase => .clase
* Etiqueta Atributo => [atributo="valor"]
* Etiqueta Clase Atributo => .clase[atributo="valor"]

NOTA: La etiqueta es opcional
